# Визуализированный консольный робот

✅Цель - вернуться в начало

⬆Программа визуализирует ваши шаги и считает их

U - вверх
D - вниз
R - вправо
L - влево

*чтобы робот "ходил" надо ввести соответствующую букву направления и нажать Enter
